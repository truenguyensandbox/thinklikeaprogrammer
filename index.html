Welcome!

Learn the basics of programming using fun, colorful blocks and real code concepts.

Variables

Variables store data values using let, const, or var.

    let name = "Ada";
    console.log(name);

Loops

Loops let you run code multiple times. A for loop repeats code for a specified number of iterations.

    for (let i = 0; i < 5; i++) {
      console.log(i);
    }

Functions

Functions are reusable blocks of code. You can define a function and call it multiple times to execute the same task.

    function greet(name) {
      console.log("Hello " + name);
    }
    greet("Ada");

Arrays

Arrays store multiple values in one variable. You can access elements using indices.

    let colors = ["red", "blue", "green"];
    console.log(colors[0]);

Objects

Objects group related data and functions together. They store data in key-value pairs.

    let person = {
      name: "Ada",
      greet: function() {
        console.log("Hi " + this.name);
      }
    };
    person.greet();

Events

Events allow you to trigger actions when users interact with elements on the page (like clicking a button).

    document.getElementById("myBtn").addEventListener("click", function() {
      console.log("Button clicked!");
    });

Conditionals

Conditionals allow you to execute code based on certain conditions. If a condition is true, one block of code is executed; otherwise, another block is executed.

    let age = 18;
    if (age >= 18) {
      console.log("You're an adult.");
    } else {
      console.log("You're a minor.");
    }

Boolean Logic

Booleans are true/false values. Use logical operators like && (and), || (or), and ! (not) to combine them.

    let hasID = true;
    let isSober = false;

    if (hasID && isSober) {
      console.log("You can enter.");
    } else {
      console.log("Access denied.");
    }

Data Types

Data types specify what kind of value a variable holds. Common types include strings, numbers, booleans, arrays, and objects.

    let name = "Ada";         // string
    let age = 31;              // number
    let isProgrammer = true;   // boolean
    let tools = ["JS", "HTML"]; // array
    let user = { name: "Ada" }; // object

Debugging

Debugging helps you find and fix errors in your code. Here's an example of how to fix a missing quote error.

    // Bug: missing closing quote
    let message = "Hello;
    console.log(message);

    // Fixed:
    let message = "Hello";
    console.log(message);

Code Comments

Code comments are used to explain what your code does. Single-line and multi-line comments help others understand your code.

    // This is a single-line comment
    let user = "Ada";

    /*
      This is a multi-line comment.
      It explains more complex code.
    */

Operators

Operators perform operations like addition, subtraction, multiplication, and comparisons.

    let a = 10;
    let b = 5;

    console.log(a + b);   // 15
    console.log(a === b); // false
